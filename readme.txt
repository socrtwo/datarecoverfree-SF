Steps to install script in server:
==================================
1. Create a mysql database
2. Import the sql file inside sql folder in the created database
3. Now copy all the files inside the code into your server.
4. Then edit the configuration file named configure.php inside the includes folder with the database details and url details.
5. Also edit the configuration file named configure.php inside the admin/includes folder with the database details and url details.
6. Thats it. All done.
